# Guide de Contribution - IRIS JE

## 🚀 Workflow Git

### 1. Créer une branche

```bash
git checkout main
git pull origin main
git checkout -b feature/US-XX-nom-de-la-branche
```

### 2. Commits atomiques

```bash
git add [fichiers]
git commit -m "feat(module): description précise"
```

### 3. Pousser et ouvrir une PR

```bash
git push origin feature/US-XX-nom-de-la-branche
```

---

## 📝 Convention de commits

| Préfixe | Usage |
|---|---|
| `feat` | Nouvelle fonctionnalité |
| `fix` | Correction de bug |
| `docs` | Documentation |
| `style` | CSS/UI |
| `refactor` | Refactorisation |
| `chore` | Configuration |
